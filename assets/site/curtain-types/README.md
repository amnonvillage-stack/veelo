# Curtain-type product photos

Drop the four product photos here. They appear on the curtain-type selection cards.

| File | Type |
|---|---|
| `eyelet.jpg` | Eyelet / ring-top curtain |
| `pleated.jpg` | Pinch-pleated curtain |
| `roman.jpg` | Roman blind |
| `roller.jpg` | Roller blind |

**Tips:**
- Photos on a dark/neutral background work best — the card frame provides the contrast.
- AVIF/WebP accepted too; just match the filename in `CurtainType.jsx` → `CURTAIN_TYPES[].photo`.
- Ideal size: 400 × 600 px, ≤ 80 KB (AVIF) / ≤ 120 KB (WebP/JPEG).

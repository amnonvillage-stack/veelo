Design decisions
Navigation
Bottom tab bar, 3 items. Home / active context / Settings. The center tab morphs based on the current step (Edit in Step 2, Catalog in Step 3, Results in Step 4). No hamburger menus — all primary actions within thumb reach.
Color system
Purple #7c6ef0 = primary actions, window tool.
Amber #f0b030 = curtain zone tool + quote/commerce signals.
Dark surface stack (#111 → #1e1e24) consistent with POC. Maintains focus on the room photo.
Generation UX
Tap "Generate 2" → both jobs run in parallel. A skeleton/spinner overlay shows per-result progress. Results appear as they complete — no waiting for all. Swipeable cards use the existing Gemini pipeline unchanged.
Catalog data model
MVP: /catalog/products.json — fields: id, name, type, price_per_m, swatch_path, collection, in_stock.
Backend serves /catalog?type=eyelet. Swatch image used directly as Gemini fabric input — no upload step needed.
Quote calculation
Fabric meters = (zone_width_cm / 100) × fullness_factor × (zone_height_cm / 100).
Fullness: eyelet 2.0×, pleated 2.5×, roman 1.1×, roller 1.0×.
Displayed on Results screen. Passed to quote endpoint.
Tech stack
Frontend: React PWA (Vite). Touch-rewrite of canvas tool with Pointer Events API.
Backend: existing FastAPI — add /catalog endpoint + parallel job queue.
State: Zustand or simple context. No auth for MVP.
